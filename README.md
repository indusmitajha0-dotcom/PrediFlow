# PrediFlow – AI-Driven Menstrual Health Tracker

## Project Overview
PrediFlow is a rule-based intelligent menstrual cycle prediction system developed for Phase-2 of Project Based Learning (PBL).

The system:
- Predicts next menstrual cycle date
- Adjusts prediction based on symptoms
- Detects anomaly patterns
- Provides confidence level and explanation

## Current Status (Mid-Term Evaluation)
- Backend API implemented using FastAPI
- Rule-based prediction logic completed
- Symptom-based anomaly detection implemented
- Frontend integration planned for end-term evaluation

## Tech Stack
- Python 3.12
- FastAPI
- Uvicorn

## How to Run

1. Create virtual environment:
   python -m venv venv

2. Activate:
   venv\Scripts\activate

3. Install dependencies:
   pip install -r requirements.txt

4. Run server:
   uvicorn main:app --reload

Open in browser:
http://127.0.0.1:8000/docs
